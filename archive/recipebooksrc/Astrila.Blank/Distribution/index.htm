Introducing DSS

Data Syndication Services or DSS specifications lets you quickly build amazing solutions with minimal effort and enable seamless collaboration in your applications.

Information on DSS specs and algorithms coming soon! Email shital@ShitalShah.com for more information.

You can,
  • Download DSS reference application and play with it.
  • Download DSS reference application source code and learn how to use Plateform APIs included in Blank.DLL (Microsoft .Net 1.1) to build your own applications.
  • Play with sample webservice which lets a web server act as an always ON endpoint and also demonstrates one way to enable two way feeds instead of just publisher-sunscriber model implicated by RSS.
  • View the sample live DSS feed for the reference application. Please note that XML namespaces aren't included yet in this feed.
